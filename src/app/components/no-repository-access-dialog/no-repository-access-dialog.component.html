Access Restricted

To view code, you need to have access to Bitbucket. To request access, fill out the access request form, selecting the following options:
  1. Select an Account
  2. Select Add role
  3. Select basic applications (LOS training)
  4. Select access to be granted: Bitbucket User

This request will be routed to your manager for approval. Once approved you may receive a training (if not done before) and after it you will be able to access the repository. Cancel Request Access
